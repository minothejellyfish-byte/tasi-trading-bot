Phases

Phase 1: LOGIN (Green) - Manual by user, capture by Mino
Phase 2: MAINTAIN (Blue) - Automated by Session Manager
Phase 3: RECOVERY (Orange) - Shared user + bot

Conflict Prevention Rules

  • Session Manager never executes trades
  • bot.py validates session before every trade (Phase 3)
  • bot.py is executor for trades (never manages tokens directly)
  • Bot never manages tokens directly (requests from SM)
  • WS Keepalive never touches auth
  • Watchdog never takes action (logs only)
